    Unity SDK 정보¶

    • 지원 가능한 최소 OS 버전은 Unity SDK에 포함된 Android, iOS SDK의 최소버전과 동일합니다.

      • iOS: iOS SDK버전 0.25.8부터는 최소 지원 iOS버전은 iOS 8.0입니다.
      • Android: Android SDK 버전 3.0부터는 안드로이드 API LEVEL 16 (Android 4.1, 4.1.1 JELLY_BEAN) 이상을 지원하고 있습니다.
