# 브랜치 Data Feeds

Data Feeds는 브랜치 시스템내의 데이터를 익스포트(Export)를 위한 툴의 모음입니다.
브랜치 데이터를 프로그램을 통해서 익스포트하거나 타 분석서비스로 데이터를 전송하는 기능을 제공합니다.
구매를 통해서 제공되는 프리미엄 제품입니다.
